# 🧠 Memory Game

A responsive and interactive Memory Card Matching Game built using **HTML, CSS, and JavaScript**. Flip the cards, find matching pairs, and complete the game in the fewest moves and shortest time.

---

## 🎮 Features

- 🎴 Randomly shuffled cards every game
- 📱 Fully responsive design (Mobile, Tablet & Desktop)
- ⏱️ Timer
- 🔢 Move counter
- 🏆 Best score tracking
- 🎯 Multiple difficulty levels (Easy, Medium, Hard)
- 🔄 Restart game option
- 🎉 Win popup
- ✨ Smooth animations
- 💾 Local storage support for best score

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🎯 How to Play

1. Choose a difficulty level.
2. Click any two cards.
3. If both cards match, they remain open.
4. If they don't match, they flip back.
5. Match all pairs to win the game.
6. Try to finish with fewer moves and less time.
